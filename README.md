# 任务记事本

##从这里开始记录生活的点点滴滴~~~

>任务记事本可以让大家很方便的记录自己任务或者计划，并且是以时间轴的方式显示，同时支持记录任务时的日期和时间功能。
>支持自动定位获取天气预报的功能,拥有很好的UI设计,给你一种亲近大自然的感觉。
